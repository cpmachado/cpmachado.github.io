***Teorema***:
$$326^2 - 325^2 \equiv 0 \text{ mod } 3$$


***Demonstração***:

$$326^2 - 325^2 = (326 + 325) (326 - 325) = 651 = 217 \cdot 3 \equiv 0 \text{ mod } 3$$
